本次「从第一性原理」做了哪些优化？
  • 只留两种并存布局：瀑布流（泛览/视觉）与分组折叠（归类/整理），共用同一套卡片组件，视觉零撕裂。
  • 移动优先（Mobile-First）：基准样式按手机写，≥480 才升两列，≥768 分组升两列；桌面预览居中限宽 920px。
  • 拇指区原则：新建按钮放右下悬浮（FAB，54px），底部固定 tab 栏，主操作不跑到屏幕顶端。
  • 触控 ≥44px：所有按钮/输入框高度达标，避免误触。
  • 零 JS 瀑布流：用 CSS columns + break-inside:avoid，不引第三方库，省电省流量。
  • 安全区适配：env(safe-area-inset-*) 照顾刘海/全面屏，顶栏与 tab 不被遮挡。
  • 无障碍：尊重 prefers-reduced-motion，语义结构 + 焦点态，缩放至 200% 不破版。
